## What WebBook is for

WebBook is a markdown-first publishing system for long-form notes and books. It combines:

- a public reading interface
- a private authoring desk
- wiki links and backlinks
- MathJax rendering
- code blocks, including runnable Python
- filesystem-backed content with revisions and exports

If you are new to the project, start with [[doubly-reinforced-coupling-beam/about-webbook]], then move to [[core-functions]] and [[webbook-handbook/installation-guide]].
